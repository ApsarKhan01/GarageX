# 🚗 GarageX - Smart Vehicle Management Dashboard

GarageX is a modern vehicle management dashboard that helps users organize and monitor every aspect of their vehicle in one place. It provides an intuitive interface for tracking fuel expenses, service history, important documents, maintenance reminders, and overall vehicle analytics.

---

## ✨ Features

- 🚘 Vehicle Profile Dashboard
- ⛽ Fuel Tracking & Expense Management
- 🔧 Service History Timeline
- 📄 Vehicle Document Management
- 🔔 Maintenance & Renewal Reminders
- 📊 Dashboard Analytics
- 📱 Fully Responsive Design
- 💾 Local Storage Support
